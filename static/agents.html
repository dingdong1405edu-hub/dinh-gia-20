Pipeline 8 Agents

Tài liệu nội bộ — mỗi agent là một LLM call (Anthropic Opus 4.7) hoặc tính toán thuần số, chạy theo thứ tự / song song. Tổng thời gian thường 180–300s/file BCTC.

Tuần tự = phải đợi agent trước Song song = chạy đồng thời với agent khác Render = không gọi LLM, chỉ vẽ PDF/Excel extended thinking tính toán thuần
[Upload BCTC] │ ▼ [A1] Extractor ──────► financials (BCĐKT, KQKD, LCTT, công ty, kỳ) │ ├──────► [A2] Industry ─┐ ├──────► [A4] Ratios ├─► [A3] Business Profile │ │ │ ▼ ├────────────────────► [A5] Projector (5Y) │ │ │ ▼ │ [A6] Valuator (DCF + Multiples + Sensitivity) │ │ │ ▼ │ [A7] Thesis Writer (Investment Memo) │ │ │ ▼ └────────────────────► [A8] Renderer ──► PDF + Excel + Debug
